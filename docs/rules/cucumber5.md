# 5本のきゅうり - ゲームルール仕様

この文書は、現行Web実装とテストに合わせたゲームルールの正本です。

## 基本情報

- プレイ人数: 2-6人
- カード: 1-15を各7枚
- デッキ枚数: 105枚
- 初期手札: 各プレイヤー7枚
- 目的: きゅうりを規定本数以上集めないこと
- 標準敗北ライン: 5本

## 用語

- ラウンド: 各プレイヤーに7枚配ってから、手札がなくなるまでの単位
- トリック: 全員が1回ずつ行動する単位
- リード: トリックの最初にカードを出すこと
- 場のカード: 現在そのトリックで勝っているカード値
- 捨てカード: 場のカードがあるときに手札の最小カードとして出せる、勝ち判定に参加しないカード
- きゅうり: 失点

## ラウンド進行

1. 105枚のデッキをシャッフルする。
2. 各プレイヤーに7枚ずつ配る。
3. 初回リードプレイヤーを決める。
4. リードプレイヤーから時計回りにカードを出す。
5. 全員が行動したらトリックを解決する。
6. 通常トリックでは、トリック勝者が次のリードになる。
7. 最終トリックでは、勝者がカード値に応じたきゅうりを受け取る。
8. 誰かが規定本数以上のきゅうりを持ったらゲーム終了。
9. まだ誰も敗北ラインに達していなければ、次ラウンドを始める。

## 通常トリック

### リード

リードプレイヤーは手札から任意の1枚を出せます。

### フォロー

場のカードがある場合:

- 場のカード以上のカードを出せます。
- 代わりに、手札の最小カードを捨てカードとして出すこともできます。
- 捨てカードは勝ち判定に参加しません。

### 勝者判定

- 勝ち判定に参加するカードのうち、最も数字が大きいカードを出したプレイヤーが勝者。
- 同じ数字が複数ある場合は、後に出したプレイヤーが勝者。
- 捨てカードは勝者判定に参加しない。

## 最終トリック

最終トリックでは、各プレイヤーが最後の1枚を出します。

- 最終トリックでは捨てカード扱いにしない。
- 通常のフォロー制限よりも、全員の最後のカードを公開して決着することを優先する。
- 勝者判定は通常と同じく、最大値、同値なら後出し勝ち。

## きゅうり本数

カードごとの基本きゅうり本数:

| カード | きゅうり |
|---:|---:|
| 1 | 0 |
| 2-5 | 1 |
| 6-9 | 2 |
| 10-13 | 3 |
| 14 | 4 |
| 15 | 5 |

## 最終トリックのペナルティ

1. 最終トリック勝者のカード値から基本きゅうり本数を決める。
2. 最終トリックにカード `1` が1枚でも含まれていれば、ペナルティを2倍にする。
3. 全員が `1` を出した場合は、ペナルティを0にする。

例:

- 勝者カードが `10` の場合: 3本
- 勝者カードが `10` で、場に `1` がある場合: 6本
- 全員が `1` の場合: 0本

## CPU難易度の方向性

### Easy

- 合法手からランダム寄りに選ぶ。
- 最終トリックの危険を強く考慮しない。

### Normal

- 最終トリックで高いペナルティを避ける。
- 中盤は低いカードの温存と処理をある程度考慮する。

### Hard

- 残りカード、順番、将来の危険度を考慮する。
- 先読みやカードカウントを強化する。
